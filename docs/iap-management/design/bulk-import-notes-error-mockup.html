Expandable Notes cell — collapsed & expanded states

ExpandableErrorCell · wired into the Apple Bulk Import result table (Step 4) only. Reusable component — Google Bulk Import can adopt later with no rewrite.

  • Collapsed: human-readable summary parsed from Apple's error (detail → title → code → raw fallback) + a Detail text button.
  • Expanded: full pretty-printed Apple error JSON, max ~10 lines then scrolls, Close collapses.
  • Each row toggles independently — expanding one does not collapse others.
