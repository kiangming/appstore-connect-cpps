Export options

Choose which countries & currencies to include in the exported file.

Same dialog for Google IAP Management and Apple IAP Management. Each country has exactly one store currency — pick the country, its currency comes along.

|

Showing 39 representative countries for this mockup — the production dialog lists all ~175 store territories, scrollable + searchable exactly like this.

Default state = every country selected (preserves today's export-everything behavior — nothing regresses for anyone who just clicks Export immediately). Deselecting narrows the xlsx's territory price columns to the chosen subset; fixed columns (Product ID/Name, Status, localizations, …) are unaffected — this dialog only controls the per-territory (Price, Currency) columns. Region grouping (Asia / Europe / Americas / Africa / Middle East / Oceania) is a scan aid over the ~175-row real list, not a selection unit — selection is always per-country.
